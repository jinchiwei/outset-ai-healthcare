# Capstone kit: Skin lesion triage

7-class: classify dermatoscopic images of skin lesions (including melanoma).

## What's here
- `starter.ipynb` -- a runnable baseline (transfer-learning ResNet18). Open it in Colab,
  run top to bottom, then improve it.

## Dataset
Uses **dermamnist** from MedMNIST (pip-installable, downloads in seconds, no account needed).

## Your goal
Beat the baseline and understand how. The rubric rewards honest evaluation and finding a
failure mode, not just a higher number.

## Level up (for ambitious pairs)
Swap in the full HAM10000 dataset (Kaggle), then try running your model on a phone photo of a mole. Watch how it behaves on data that looks nothing like training.
